Thank you for your interest in contributing to PyCypher!

For the full contributing guide — including development environment setup, coding standards, testing, and pull request workflow — see:

## Quick Start

```bash
git clone https://github.com/zacernst/pycypher-nmetl.git
cd pycypher-nmetl
make setup
make check
```
