Overview
--------

The NMETL project consists of four main packages:

* **nmetl**: The core ETL framework providing declarative data transformation
* **pycypher**: Cypher query parsing and graph database functionality
* **shared**: Shared utilities and helper functions
* **fastopendata**: Data source integrations for common open data formats
