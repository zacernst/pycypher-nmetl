NMETL
=====

NMETL (New Methods for ETL) is a declarative ETL framework that leverages the PyCypher parser to enable a novel approach to data transformation and loading.

Features
--------

* Declarative data modeling
* Fact-based storage
* Trigger-based transformations
* Format-agnostic data representation
* Configuration via YAML

Components
----------

The NMETL package consists of several key components:

* **Session**: The central orchestrator for ETL processes
* **Data Sources**: Connectors for various data sources
* **Triggers**: Declarative transformations using Cypher queries
* **Configuration**: YAML-based configuration system
