pycypher-nmetl documentation
============================

Declarative ETL using Cypher queries
-----------------------------------

**pycypher-nmetl** is a Python library that provides a declarative approach to ETL (Extract, Transform, Load) operations using Cypher queries for Neo4j databases.

Features
--------

* Declarative ETL pipelines using Cypher queries
* Integration with Neo4j databases
* Support for complex data transformations
* Extensible architecture
